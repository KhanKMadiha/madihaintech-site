What this chatbot does

This chatbot is designed as a lightweight portfolio assistant. It can answer common questions about my background, highlight the areas I work in most often, and point visitors to relevant sections across my site.

You can try it any time using the chat bubble in the bottom right corner.

Support workflows Conversation design Intent routing Portfolio assistant

How I built it

  • 1. Learning foundation

    IBM AI Foundations learning path on Coursera

    I completed the IBM AI Foundations learning path and used it to ground my understanding of core AI concepts and practical use cases.

  • 2. First build

    IBM Watson chatbot prototype

    I built an initial chatbot using IBM Watson to explore dialogue structure, intents, and confidence-based responses.

  • 3. Iteration and deployment

    Dialogflow assistant integrated into my site

    I then applied what I learned to implement a Dialogflow chatbot and embedded it directly into this portfolio so visitors can explore it in context.

What I am improving next

I am iterating on response quality, navigation accuracy, and helpfulness. The goal is to make it feel like a real support assistant, not a gimmick, and to keep refining how it guides users to the right information.

Better fallback handling More guided navigation Higher quality responses Project documentation
